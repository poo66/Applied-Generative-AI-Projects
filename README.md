# LangChain Text Generation using Hugging Face FLAN-T5

## Overview

This project demonstrates text generation using:

- Hugging Face Transformers
- Google FLAN-T5 Large Model
- LangChain
- Prompt Templates
- LLMChain

The application generates step-by-step answers for user questions using a pre-trained language model.

---

## Technologies Used

- Python
- Hugging Face Transformers
- LangChain
- PyTorch
- FLAN-T5 Large
- Pytest
